o   Introduction

IBA_J was developped in order to 1) enable multiple data file conversion and 2) use the image analysis capabilities offered by ImageJ. The latter is the main reason why this plugin is developed under JAVA.

Installation

To install the plugin, it is imperative to first install imageJ.

Then, unzip the plugin file in the \plugins directory.

Copy the IJ_prefs.txt and IJ_prop.txt files in the app directory (the one containing ij.jar)

IBA_J Users guide

Treatment of AIFIRA data requires 2 main steps:

        Conversion of listfiles (raw data) into individual detector files. Listfiles are generated during acquisition after saving all detected events (eg. Emitted particles, photons, scattered particles) in chronological order. In listfiles, events detected by different detectors are the mixed into one single file. This feature is interesting to manage coincidence and perform dynamic analysis. For imaging nevertheless, information corresponding to a single detector type is usually interesting.

        Mapping calculation and spatial ROI definition for a single detector type
